# Comunidad SV - Gestor de Eventos

## Pasos para ejecutar

git clone https://github.com/tu-usuario/app-eventos.git
cd app-eventos

## Sino directamente desde la rama siguiente:

git clone -b jorge-diaz https://github.com/IAvalon-boy/app-eventos.git

## instalación de dependencias

npm install
npx expo install expo-linear-gradient expo-notifications firebase
npx expo start -c

Luego escaneamos el código QR que aparece en consola en ese momento y lo desplegamos en la aplicación Expo Go para android
